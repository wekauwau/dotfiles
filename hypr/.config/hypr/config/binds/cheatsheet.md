`mainMod`: `SUPER`
`shiftMod`: `SUPER + SHIFT`

---

`CTRL + M`: Quit hyprland

# App

## TUI

`mainMod + RETURN`: Terminal
`shiftMod + RETURN`: Floating terminal
`mainMod + O`: File manager
`shiftMod + O`: Floating file manager
`mainMod + J`: Task manager (htop)
`shiftMod + J`: Task manager (btop)
`mainMod + N`: Text editor
`shiftMod + N`: Floating text editor

## GUI

`mainMod + SLASH`: Password manager

# Media

`XF86AudioRaiseVolume`: Volume +5%
`ALT + XF86AudioRaiseVolume`: Volume +1%
`XF86AudioLowerVolume`: Volume -5%
`ALT + XF86AudioLowerVolume`: Volume -1%
`XF86AudioMute`: Toggle audio mute
`XF86AudioMicMute`: Toggle mic mute
`XF86MonBrightnessUp`: Brightness +5%
`ALT + XF86MonBrightnessUp`: Brightness +1%
`XF86MonBrightnessDown`: Brightness -5%
`ALT + XF86MonBrightnessDown`: Brightness -1%

# Tiling

`shiftMod + L`: Cycle layout for current workspace (monocle, **scrolling**)
`mainMod + Q`: Toggle adios workspace
`shiftMod + Q`: Move window to adios workspace
`mainMod + E`: Toggle amogus workspace
`shiftMod + E`: Move window to amogus workspace
`mainMod + [0-9]`: Switch workspaces
`shiftMod + [0-9]`: Move active window to a workspace
`mainMod + TAB`: Switch to next active workspace
`shiftMod + TAB`: Switch to previous active workspace
`mainMod + GRAVE`: Switch to last opened workspace
`ALT + TAB`: Switch to next window
`ALT + SHIFT + TAB`: Switch to previous window
`mainMod + W`: Switch to up window
`mainMod + S`: Switch to down window
`mainMod + A`: Switch to left window
`shiftMod + A`: Swap left
`mainMod + D`: Switch to right window
`shiftMod + D`: Swap right
`mainMod + U`: Close window
`shiftMod + U`: Kill window
`mainMod + P`: Toggle floating window
`mainMod + F`: Toggle maximized window
`shiftMod + F`: Toggle fullscreen window
`mainMod + G`: Group/ungroup next
`shiftMod + G`: Group/ungroup previous
`mainMod + EQUAL`: Window size++
`mainMod + MINUS`: Window size--

# Utilities

`mainMod + SPACE`: Menu
`mainMod + I`: Open notification center
`CTRL + ALT + SHIFT + W`: Restart walker
`Print`: Screenshot (monitor)
`mainMod + Print`: Screenshot (window)
`shiftMod + Print`: Screenshot (region)
